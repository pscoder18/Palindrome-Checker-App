### Use Case 5 (UC5): Stack Based Palindrome Checker
* **Goal:** Use a Stack data structure to reverse characters and validate if a string is a palindrome.
* **Key Concepts:** * **Stack Data Structure:** A linear data structure that operates on the Last-In-First-Out (LIFO) principle.
    * **Push Operation:** Inserting characters into the stack one by one.
    * **Pop Operation:** Removing characters from the stack. Because of LIFO, popping retrieves the characters in reverse order.
    * **Reversal Logic:** Maps the natural behavior of a stack to the string reversal process required for palindrome validation.
* **Data Structure:** `Stack`
